Redis

Posted on 2020-08-07 | Post modified: 2020-08-07 | In Spring

Redis

非关系型数据库，缓存数据库，有点类似于mongodb不过是基于内存进行存储

数据存储在内存中而不会存在硬盘的文件中，结构是K-V对，没有数据表的概念，只有一个整体的字典

持久化存储，即使Redis服务关闭了，数据也在内存中

底层使用C语言编写，Java可以通过Spring Boot调用Redis

支持数据类型

字符串、列表、集合、有序集合、哈希

除了哈希其他的存储形式都是{key:value}，哈希是{key:{key:value} }即通过key找到一个map

安装

下载安装包，解压

之后任意位置新建一个文件夹，包括

  • bin：放置启动Redis的可执行文件
  • db：放置数据文件
  • etc：放置配置文件，设置端口、bin文件夹位置、db文件夹位置
  • 以及一个日志文件

启动

启动服务：在新建的文件夹中运行命令./bin/redis-server ./etc/redis.conf

启动客户端管理窗口：./bin/redis-cli

存取数据

相当于只在一个表里，直接存或者取就行了

key一般都是String类型

存储：set key value

获取：get key value

删除：del key

查询所有的key：keys *

Spring Boot整合Redis

实际是在Spring Boot中用Spring Data Redis框架操作Redis

在maven工程中添加spirng-boot-starter-data-redis和commons-pools2依赖

在application.properties中配置Radis

实体类对象存入Redis

实体类需要实现序列化接口Serializable，一个对象存储起来都要实现序列化接口

例如在Controller里使用Redis

存入Redis后，由于序列化操作，Redis会自动给key和value中每一个值前面加上一段前缀

取出的时候，Redis也会自动进行反序列化

五种数据类型存入Redis

字符串存取
列表存取
集合
有序集合
哈希

redis中，通过一个outerKey找到HashMap，再在HashMap中用key找value
