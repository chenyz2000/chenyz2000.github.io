C++注意点

Posted on 2020-01-13 | Post modified: 2020-01-13 | In nec

数据类型

0加在整数前表示八进制数字

auto定义变量，根据初始化的值推断

static修饰局部变量可以保持其值，修饰全局变量可以使其作用域限制在声明它的文件里

extern修饰全局变量使之对整个程序可见

函数传参

  • 将变量名作为实参和形参。这时传给形参的是变量的值，传递是单向的。如果在执行函数期间形参的值发生变化，并不传回给实参。因为在调用函数时，形参和实参不是同一个存储单元。//同c

  • 传递变量的指针。形参是指针变量，实参是一个变量的地址，调用函数时，形参(指针变量)指向实参变量单元。传递一个指针给函数，可以在函数内改变指针指代的值。//同c

  • C++提供了传递变量的引用。形参是引用变量，和实参是一个变量，调用函数时，形参(引用变量)指向实参变量单元。这种通过形参引用可以改变实参的值。

指针

int * ptr[3]相当于int *(ptr[3])，是指针的数组

int (*ptr)[3]是指向数组的指针

如const char *names[MAX] = { “Zara”, “Hina”, “Nuha”, “Sara”};指针数组，存储的是各个字符串数组的首地址，cout时输出字符串

指向结构的指针struct Books *struct_pointer;

指针访问结构的成员，必须使用 -> 运算符struct_pointer->title;

通过对象访问结构的成员时使用点运算符，通过指针访问结构的成员时使用箭头运算符。

类与结构体

类与结构体在 C++ 中只有以下几点区别，除此这外无任何区别。

  • class 中默认的成员访问权限是 private 的，而 struct 中则是 public 的。

  • 从 class 继承默认是 private 继承，而从 struct 继承默认是 public 继承。

  • class 可以定义模板，而 struct 不可以。
