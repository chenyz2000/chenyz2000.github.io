MyBatis

Posted on 2020-08-05 | Post modified: 2023-01-30 | In Spring家族

MyBatis

MyBatis是ORM框架

ORM，Object Relational Mapping，对象关系映射。三个单词分别指Java的对象，数据库的关系，和映射

MyBatis是ORM框架，是基于Java的持久层框架，实现了数据持久化

连接Java程序和数据库，使Java开发者可以用面向对象的思想来管理数据库，支持通过配置或注解，将接口和POJO映射成数据库中的记录

是对JDBC的封装，相对于传统的JDBC，简化了连接、操作的过程，减少了一半的代码量，只需要把关注点放在SQL的增删改查上

除了MyBatis以外，还有Hibernate、Spring Data。MyBatis是半自动的，Hibernate是全自动的，Spring Data是基于注解的

Spring Boot

Posted on 2020-07-11 | Post modified: 2022-05-29 | In Spring家族

Spring Boot

官方文档

简化Spring，将其中复杂的XML配置Bean的部分砍掉，是Spring的再封装

核心思想：CoC约定大于配置，Spring Boot已经配置好了约定俗成的部分，开发人员只需要定制不符合约定的部分

Spring Boot使得易于集成，如集成数据库、缓存、权限、web容器、监控、消息队列

原本开发Web项目需要：配置maven依赖，配置web.xml加载Spring，配置数据库连接配置Spring事务，配置加载配置文件的读取开启注解，配置日志文件，…，部署Tomcat进行调试

而使用Spring Boot创建即可使用：建立一个Spring Boot工程，配置maven依赖，接可以直接启动

Spring Boot配合Spring Cloud：Spring Boot用于开发微服务中的一个个服务，SpringCloud用于管理Spring Boot开发的多个服务
