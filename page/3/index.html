websocket

Posted on 2021-04-12 | Post modified: 2021-07-04 | In 其他

含义

websocket是一种网络通信协议，基于TCP，类似HTTP

HTTP只能客户端向服务器发送消息，要想客户端获得服务器的消息就得是长轮询

websocket实现了客户端与服务器全双工通信，握手之后建立长时间的连接，可以由服务器主动发送信息通知客户端

服务器和客户端都可以既监听消息也发送消息

python、Java、Node都有Websocket服务端的库

在Java中大多使用双线程实现全双工，一个发一个收

Python os库

os.path子模块

即如os.path.exists()，可以通过from os import path导入此模块

  • exists(path)，其中path为文件或目录路径字符串

  • abspath(path)，获得文件绝对路径

  • isfile(path)，isdir(path)

  • getsize(path)，返回文件大小，以字节为单位

  • basename(path)返回不含目录的文件名，dirname(path)返回不含文件名的目录名，split(path)返回目录名和文件名的二元组，splitext(path)返回路径加文件名和扩展名的二元组

  • join(path, [path1, …])合并路径

Linux图形界面

Posted on 2021-03-16 | Post modified: 2021-06-09 | In Linux

:heavy_check_mark:图形化接口协议

  • SSH+X11转发（首选）
    Linux系统启用X协议，配置X Client，本地有桌面环境的机器（如Windows）配置X Server
  • RFB：Remote Frame Buffer，VNC常用的协议（推荐）
    需要Linux系统安装X Windows图形界面
  • Xdmcp：透过X display manager（xdm, gdm, kdm等）所提供的功能协议
  • RDP：Remote Desktop Protocol，windows系统的默认协议
  • 其他：FreeNX，Xpra，向日葵，teamviwer

xmanager支持XDcmp和XStart（SSH+X11）两类会话

vnc是远程连入操作系统，所有操作在UNIX、Linux主机服务端进行，即使操作中网络断开，也不影响操作的进行

xmanager通过端口将主机服务器的UI界面引导到本地电脑进行展现，如果操作中网络断开，操作将失败

vnc在远程端上占用更小，xmanager更清晰

概述

问题一 存储

2000年，谷歌意识到数据大量增长，从TB增长到PB，单机难以存储

于是谷歌提出分布式文件系统(DFS)

把文件分成多个文件块，并且每个文件块可以放在若干个机器上有多个备份，实现容灾

使用者直接使用DFS的接口，不需要关注文件块的底层存储和查找细节
