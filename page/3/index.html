npm

Posted on 2021-02-01 | Post modified: 2021-02-15 | In Nodejs

npm

NPM，node package manager，是一个node.js的包管理器（类似linux的apt）

我们在传统开发的时候，jQuery.js大多都是百度搜索，然后去官网下载，或者直接引入CDN资源，这种方法太过于麻烦。如果以后遇到其他的包，这个包的代码本身可能还调用了其他的包（即存在依赖关系），那么我们要在自己的项目中引入一个包将变得十分困难。现在我们有了NPM这个包管理器，直接可以通过npm install xxx包名称的方式引入它，比如说npm install vue

作为一个前端开发，即使你不用Node.js写独立程序，也得配一个Node.js运行环境，毕竟很多前端工具都是使用它写的。

Nodejs

Posted on 2021-01-28 | Post modified: 2021-01-28 | In Nodejs

简介

Node.js是一个能在服务器端运行JS代码的开源的、跨平台的JS运行环境

从浏览器到服务器，原本只可以对浏览器进行操作，现在可以用JS独立写程序对系统进行操作，让JS和Java、C等编程语言地位一样了

JS标准定义的API只为构建浏览器的应用程序，而commonJS制定了一个更广泛的标准库，类似Java、Python的规范，使JS不仅仅停留在浏览器脚本的阶段

CommonJS是模块化的标准，Nodejs是CommonJS的实现。

Posted on 2020-07-06 | Post modified: 2020-12-21 | In Java Web

J2EE 框架

  • 无框架传统Web开发模式
    JSP业务逻辑和显示+Java Bean模型

  • 早期的MVC模式
    JSP视图View+Servlet控制器+Java Bean模型

  • 第一代框架集SSH
    包括基于MVC的Web框架Struts2+容器框架Spring+持久层框架Hibernate
    Struts2做前端控制器，Spring管理各层的组件，Hibernate负责持久化层

  • 第二代框架集SSM
    包括基于MVC的Web框架Spring MVC+容器框架Spring+持久层框架Mybatis
    SpringMVC做前端控制器，Spring管理各层的组件，MyBatis负责持久化层

  • Spring Boot

  • Spring Cloud
