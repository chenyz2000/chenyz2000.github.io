websocket

Posted on 2021-04-12 | Post modified: 2021-07-04 | In 其他

含义

websocket是一种网络通信协议，基于TCP，类似HTTP

HTTP只能客户端向服务器发送消息，要想客户端获得服务器的消息就得是长轮询

websocket实现了客户端与服务器全双工通信，握手之后建立长时间的连接，可以由服务器主动发送信息通知客户端

服务器和客户端都可以既监听消息也发送消息

python、Java、Node都有Websocket服务端的库

在Java中大多使用双线程实现全双工，一个发一个收

种类

  • Bourne shell（sh）

  • C shell（csh）

  • Korn shell（ksh）

  • bash < script.sh，输入重定向，会产生新进程

  • · script.sh，在当前shell中执行（·表示当前shell），不会产生新进程
  • 将脚本设置为可执行，之后./script.sh直接执行

Python os库

os.path子模块

即如os.path.exists()，可以通过from os import path导入此模块

  • exists(path)，其中path为文件或目录路径字符串

  • abspath(path)，获得文件绝对路径

  • isfile(path)，isdir(path)

  • getsize(path)，返回文件大小，以字节为单位

  • basename(path)返回不含目录的文件名，dirname(path)返回不含文件名的目录名，split(path)返回目录名和文件名的二元组，splitext(path)返回路径加文件名和扩展名的二元组

  • join(path, [path1, …])合并路径
