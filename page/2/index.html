Java注解

Posted on 2020-05-30 | Post modified: 2021-10-05 | In Java基础

注解Annotation

格式：@注解名，有的需要加参数@注解名(xxx=...)

源码=代码+注释+注解

注解可以在包、类、方法、字段、参数等类型上添加

  • 被编译器读取，提供信息给编译器、IDE
  • 通过反射机制读取注解，在程序运行时访问，提升程序的动态性
  • 使用某些工具产生额外的代码、配置文件

快捷键

重要快捷键

ctrl + alt + L：格式化代码

ctrl + W：扩大选中的代码块，再加shift是缩小

ctrl + D：向下复制当前行

ctrl + Y：删除当前行

ctrl + O：重写、实现父类或接口的方法

alt + insert：生成getter、setter、toString等

ctrl + alt + O：调整导入的包

ctrl + alt + T：为当前代码选定环绕代码(if、for、try等)

ctrl+shift + del：删除环绕代码
