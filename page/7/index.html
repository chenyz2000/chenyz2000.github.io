Spring

Java EE领域的企业级开发框架，是软件设计层面的框架，能帮企业实现复杂的网站后端服务

分层

优势是可以将应用程序进行分层，开发者可以自主选择组件

MVC层：Structs2、Spring MVC

ORM层：Hibernate、MyBatis、Spring Data

两大核心机制

  • IoC控制反转：针对复杂的对象管理问题，将设计好的对象交给IoC容器而不是new。工厂模式。
  • AOP面向切面编程：代码解耦，不改变源代码功能流程的基础上加入新功能，扩展性。代理模式。

Python字符编码

Posted on 2020-05-29 | Post modified: 2020-05-29 | In Python

Python中的编码

编解码

文件中采用UTF-8或GBK编码方案，在Python中string采用Unicode码

UTF-8、GBK到Unicode，即读取文件到程序，使用解码s.decode("GBK")

Unicode到UTF-8、GBK，即输出程序中字符串，使用编码str.encode("UTF-8")

在Python程序中，统一采用Unicode编码进行字符串的裁剪、拼接、正则等操作。所以应该尽早将读取的字符解码为Unicode，在最后输出的时候将Unicode编码成UTF-8

Python中的string采用Unicode，而二进制格式bytes采用UTF-8、GBK等编码
