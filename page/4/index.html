JSP和Servlet

Posted on 2020-12-21 | Post modified: 2020-12-21 | In Java Web

Servlet和JSP都是SUN公司提出的解决服务器端的操作，Servlet早于JSP，JSP会转换成Servlet

JSP比较简略，早期没有JSP时，就用Java中的servlet接口

Servlet是一套用来处理网络请求的Java Web规范，JSP是建立在Servlet规范之上的Web开发技术

Servlet形式是Java接口（狭义）、Java程序（广义），JSP形式是在HTML中嵌套Java代码

Servlet和JSP都应用于服务器端（部署于Tomcat服务器），用于与数据库通信、动态生成网页

Servlet运行在Servlet容器（Tomcat，应用服务器）中

web服务器有Tomcat，Resin，Jboss，WebSphere，WebLogic

JSP在Servlet容器中由JSP引擎转化为Servlet

JSP会转化为Servlet（即翻译成Java代码），编译为class文件后执行

webpack中使用Vue

Posted on 2020-08-28 | Post modified: 2020-08-28 | In Vue

webpack导入Vue包

npm安装Vue，然后在main.js中导入Vue模块

Vue分为两个版本

  • 完整版vue/dist/vue.js
  • 非完整版vue/dist/vue.runtime.common.js，webpack默认导入

非完整版仅提供了runtime-only的方式，功能不全，不能按照注册组件的方式使用组件

导入完整版Vue

有以下多种修改方法：

  • 修改main.js中的导包import Vue form "../node_modules/vue/dist/vue.js"

  • 修改node_modules/vue/package.json，将"main":"dist/vue.runtime.common.js"改为"main":"dist/vue.js"。

    import查找规则（node的require与它一致）：找node_modules文件夹=>找包的文件夹=>找package.json包配置文件=>找main属性，main属性指定了这个包在被加载时的入口文件

  • 配置webpack.config.js
