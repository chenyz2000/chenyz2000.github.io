Linux图形界面

Posted on 2021-03-16 | Post modified: 2021-06-09 | In Linux

:heavy_check_mark:图形化接口协议

  • SSH+X11转发（首选）
    Linux系统启用X协议，配置X Client，本地有桌面环境的机器（如Windows）配置X Server
  • RFB：Remote Frame Buffer，VNC常用的协议（推荐）
    需要Linux系统安装X Windows图形界面
  • Xdmcp：透过X display manager（xdm, gdm, kdm等）所提供的功能协议
  • RDP：Remote Desktop Protocol，windows系统的默认协议
  • 其他：FreeNX，Xpra，向日葵，teamviwer

xmanager支持XDcmp和XStart（SSH+X11）两类会话

vnc是远程连入操作系统，所有操作在UNIX、Linux主机服务端进行，即使操作中网络断开，也不影响操作的进行

xmanager通过端口将主机服务器的UI界面引导到本地电脑进行展现，如果操作中网络断开，操作将失败

vnc在远程端上占用更小，xmanager更清晰

Posted on 2021-02-01 | Post modified: 2021-02-15 | In Nodejs

npm

NPM，node package manager，是一个Node.js的包管理器（类似linux的apt）

我们在传统开发的时候，jQuery.js大多都是百度搜索，然后去官网下载，或者直接引入CDN资源，这种方法太过于麻烦。如果以后遇到其他的包，这个包的代码本身可能还调用了其他的包（即存在依赖关系），那么我们要在自己的项目中引入一个包将变得十分困难。现在我们有了NPM这个包管理器，直接可以通过npm install xxx包名称的方式引入它，比如说npm install vue

作为一个前端开发，即使你不用Node.js写独立程序，也得配一个Node.js运行环境，毕竟很多前端工具都是使用它写的。
