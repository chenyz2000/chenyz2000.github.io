Redis

Posted on 2020-08-07 | Post modified: 2020-08-07 | In Database

Redis

非关系型数据库，缓存数据库，有点类似于mongodb不过是基于内存进行存储

数据存储在内存中而不会存在硬盘的文件中，结构是K-V对，没有数据表的概念，只有一个整体的字典

持久化存储，即使Redis服务关闭了，数据也在内存中

底层使用C语言编写，Java可以通过Spring Boot调用Redis

支持数据类型

字符串、列表、集合、有序集合、哈希

除了哈希其他的存储形式都是{key:value}，哈希是{key:{key:value} }即通过key找到一个map

Spring Boot

Posted on 2020-07-11 | Post modified: 2020-08-04 | In Spring家族

Spring Boot

官方文档

简化Spring，将其中复杂的XML配置Bean的部分砍掉，是Spring的再封装

核心思想：CoC约定大于配置，Spring Boot已经配置好了约定俗成的部分，开发人员只需要定制不符合约定的部分

Spring Boot使得易于集成，如集成数据库、缓存、权限、web容器、监控、消息队列

原本开发Web项目需要：配置maven依赖，配置web.xml加载Spring，配置数据库连接配置Spring事务，配置加载配置文件的读取开启注解，配置日志文件，…，部署Tomcat进行调试

而使用Spring Boot创建即可使用：建立一个Spring Boot工程，配置maven依赖，接可以直接启动

springBoot用于开发微服务中的一个个服务，springcloud用于管理springboot开发的多个服务
