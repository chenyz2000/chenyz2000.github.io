Redis

Posted on 2020-08-07 | Post modified: 2020-08-07 | In Database

Redis

非关系型数据库，缓存数据库，有点类似于mongodb不过是基于内存进行存储

数据存储在内存中而不会存在硬盘的文件中，结构是K-V对，没有数据表的概念，只有一个整体的字典

持久化存储，即使Redis服务关闭了，数据也在内存中

底层使用C语言编写，Java可以通过Spring Boot调用Redis

支持数据类型

字符串、列表、集合、有序集合、哈希

除了哈希其他的存储形式都是{key:value}，哈希是{key:{key:value} }即通过key找到一个map

注解整理

Posted on 2020-08-06 | Post modified: 2020-08-06 | In Spring家族

参考

Spring Boot注解

@Component是通用注解，@Controller、@Service、@Repository是它的拓展，都是Spring注解，注解后可以被Spring框架所扫描并注入到Spring容器来进行管理

控制层

@Controller

@ResponseBody，注解类或方法，使其转换为指定格式后写入到Response对象的body数据区

@RestController = @Controller + @ResponseBody，注解在类上

@RequestMapping有多个衍生注解，@GetMapping，@PostMapping

@RequestParam

@RequestBody

@ModelAttribute

详见《前后端分离》

@Autowired和@Resource都是自动装配

服务层

@Service

持久层

@Repository注解类作为DAO对象

配置bean

@Value(“”)为对象的属性赋值，在每个属性上注解；@ConfigurationProperties(prifix="...")，注解在类上，使用配置文件进行属性赋值，相当于简化了xml的配置文件而已

@PropertySource()读取指定的properties、yml配置文件，可以指定多个，和ConfigurationProperties啥的一起用

@ImportResource(locations={})读取xml配置文件（因为springboot默认不读取xml配置）

推荐方式：全注解。@Configuration标注配置类，相当于配置文件；@bean标注方法，将方法的返回值添加到IoC容器中，方法名即为默认的id

Spring注解

import org.springframework.beans.factory.annotation.*;

jar包位于仓库的org\springframework\spring-beans\5.x.x.RELEASE\spring-beans-5.1.5.RELEASE.jar

@Value

@Required
