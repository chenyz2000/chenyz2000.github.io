Lie Group Covering Learning Algorithm

覆盖是代数几何研究中的重要工具。在机器学习中，基于覆盖的思想可以得到多种分类手段。

背景介绍



     覆盖学习算法是一种具有高分类准确度和强泛化能力的构造性学习算法，可以处理海量数据的多种分类和聚类，在许多应用和研究领域应用广泛。

    属于监督学习，在一些研究方向上都发挥着重要的作用，如图像检索、金融预测、无线传感网络监控、文本分类、手写汉字的识别等。

    其优势在于：将李群结构分解，将其映射到各覆盖群中，能够有效地提高学习效率，丰富李群机器学习理论，同时也为数据分析提供一种新的机器学习方法。

研究意义

Question


    在学习过程中，直接对多值表示问题进行分析时，往往是困难的。

Solution


    我们可以把李群问题转化到其覆盖群上进行求解，利用李群和覆盖群之间的同构关系，有效的解决这一难题。

Specific Ideas


    在数学中可以证明，当李群G的群空间是多连通时，一定存在另一个群空间是单连通的简单李群，它与原李群G之间有多一对应的同态关系，称之为李群G的覆盖群。

    覆盖群和李群之间存在着局域同构的关系，可以同构的表示原李群的性质，这恰好对应着学习问题，通过对样本集的学习，可以部分地还原问题世界，从而解决真实问题。

    我们可以利用李群特殊的代数几何结构，对样本数据进行有效表示。样本构成空间是嵌入在一个微分流形中，构成一个李群。

    通过对李群性质的分析，得知李群的性质与其群空间的连通度有密切关系。

    利用单连通覆盖算法可以找到其覆盖群和覆盖映射，通过覆盖映射就可以将李群中的元素和覆盖群的元素建立一个一一对应的关系，就能够把李群问题转换为其覆盖群上的问题，从而将非线性问题转化为线性问题。

Question


    然而，很多情况下，学习问题并不是一一对应的，往往会有多输入空间但输出域的情况，甚至是多输入多输出。 在学习过程中，直接对多值表示问题进行分析时，往往是困难的。

Solution


     同样，可以将李群问题转化到其覆盖群上进行求解，因为李群的许多整体性质可以从群的无穷小结构推出，即群的若干个片给出后，通过对局域性质的分析，李群的整体性质就可以完全确定。

Specific Ideas


    多连通李群的群空间内的连线根据是否能够连续变化，可以分成若干组，同一组内的连线可以互相变化。这样群空间就被分成不相连结的若干片，每个片表示一个独立的部分，都可以找到各片的覆盖群。

    覆盖群由于与李群之间是局域同构的，因此覆盖群的组合函数真实地反映了多连通李群的性质。根据覆盖群这一特性，就可以把复杂的多连通问题简化为覆盖群的组合问题。

研究现状



覆盖技巧是研究代数几何的重要工具，比如在研究基变换、典范映射、奇点等问题的时候都会涉及覆盖。现在覆盖思想具有较高的准确度和强泛化能力，已经被应用到更多的领域，如数据挖掘，机器学习等等。

具体研究如下：


  1. 邻域覆盖学习算法：通过球面投影变换将神经网络的最优设计问题转换为某种最优覆盖问题；

  2. 覆盖聚类算法：采用覆盖的概念将比较集中的样本聚合在一起，从而发现隐含在样本集中的类；

  3. 核覆盖算法：将SVM中的核函数法与构造性学习的覆盖算法相融合，使得运算速度更快、精度更高、稳健型更强；

  4. 改进的流形学习方法：引用邻域覆盖思想，从计算上用有限样本集描述流形结构，从而这组有限数据点集能够覆盖嵌套流形的支撑域，不致丢失结构信息，同时尽可能的少地保持存储所需的码表。

  5. 仿生/拓扑模式识别：确认了特征空间中同类样本的连续性，以一类样本在特征空间中的分布的最佳覆盖作为目标

    由此可见，覆盖思想在机器学习领域已经得到了广泛的应用， 覆盖思想主要是把同类问题进行覆盖，从而把不同类问题分割开来。因此通过对同类样本进行学习，能够得到更好的学习能力。 在对多类问题进行拟合的时候，可以利用李群特殊的代数几何结构，最大程度的保持问题空间的不变性。

最新进展



现已有学者提出了李群覆盖学习算法，包括单连通覆盖学习算法和多连通覆盖学习算法，并且已经将其引入到药物分子设计领域，而且在构效关系模型和分子对接模型等具体应用中取得了令人满意的结果。



多连通李群覆盖学习算法在图像分类上的应用-严晨


    根据李群的连通性质，将具有不同类别信息的研究对象映射到多连通李群空间，并结合覆盖的思想，提出了多连通李群覆盖学习算法(MCLGC-LA)， 并应用该算法处理图像的分类问题。

    在两个数据集上进行了实验，结果表明多连通覆盖算法比经典李群上的均值算法(Lie-Mean)以及改进后的 Lie-Fisher 算法具有更优的分类效果，而且算法在处理手写体 MNIST 数据集时还具有较好的稳定性。

多李群覆盖学习优化算法-吴鲁辉


    从目标李群空间的性质出发，提出了一种新的方法来减少覆盖学习中的道路交叉问题。从宏观上看，由于目标李群空间上的样本更加集中，减少了道路交叉情况，同时集中使得样本的拒识现象的判别更加有效。

    对于训练而言，如何得到更好的描述图像的特征是关键。通过同态映射将原始李群映射为目标李群， 使得目标李群空间中的样本具有一定的集中性，从而减少道路交叉问题。

存在问题



鉴于学习问题本身的复杂性，覆盖学习算法还存在进一步改进和发展的地方，例如：

1. 覆盖模型的选择问题。

针对具体问题，需要选择特定的覆盖模型来模拟样本的学习过程才能最优化覆盖效果。然而由于问题的复杂性和多样性，目前还没有规范的指导依据给出选择方案，大部分情况还需要人的经验来指导。

2. 样本属性的特征优化问题。

覆盖学习对样本特征提取的要求较高，如果提取的特征较少且不具有代表性，或者提取的特征含有噪声较多，都会严重影响训练得到的覆盖模型的效果，特征较少将造成更多样本点被误判，特征噪声较多会使得被拒识的样本点增多。 然而，现实研究的对象往往具有较高的复杂性和不确定性，高效处理数据的要求迫使很多提取的样本特征是不完整和不准确的，如何保证覆盖学习的识别性能是有待于进一步研究的问题。


3. 覆盖个数的选择和优化问题

研究发现，训练得到的覆盖个数越少，机器学习的泛化能力越强，但识别精度可能有限； 覆盖个数越多，覆盖网络对各类别样本特例也会考虑，识别精度会增加，但往往会造成过拟合问题，使得泛化能力降低。 如何平衡识别精度和泛化能力之间的关系，需要进一步探讨。

4. 动态数据的学习问题

虽然已有研究者给出了覆盖学习的增量学习方法，但由于所研究数据量的急剧扩大，这些方法不可避免会暴露其局限性， 因此需要进一步研究出对数据动态更新较快的新的增量覆盖算法。
