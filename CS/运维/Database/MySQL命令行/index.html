MySQL

Posted on 2020-03-03 | Post modified: 2020-03-03 | In Database

命令行

mysqld --initialize --user=root --console 安装后初始化root账户，会产生暂时密码，会在…/data下产生自带的数据库

mysqld --console 开启mysqld进程，需要命令行保持开启，可以ctrl+c关闭mysqld

mysql -uroot -p 连接mysqld服务器

mysql> ALTER USER ‘root’@‘localhost’ IDENTIFIED BY ‘1019’ PASSWORD EXPIRE NEVER;登录后可以改密码

linux安装

mysql安装后会在/var/log/mysqld.log里写默认密码

登录mysql后需要更改密码

密码具有要求等级，默认是1级，可以改成0级——set global validate_password_policy=0;

远程访问

想从远程访问数据库，需要修改mysql数据库中的user表——update user set host = ‘%’ where user = ‘root’;

再刷新一下flush privileges;
