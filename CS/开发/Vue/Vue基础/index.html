Vue基础

Posted on 2020-07-23 | Post modified: 2020-08-24 | In Vue

Vue.js

前端框架，擅长于数据操作、表单填写、页面内容根据数据进行渲染、页面元素部分刷新

页面由数据生成，只要数据就可以进行页面视图的渲染，不需要再操作DOM元素

使用Vue时，建议不使用jQuery，尽量不要做Dom操作

前端发展过程：

  1. 原生JS
  2. jQuery等类库。解决了浏览器大部分兼容性问题
  3. 前端模板引擎。调用模板引擎的方法生成DOM元素渲染页面，而不是直接操作DOM，解决了频繁操作DOM的问题
  4. Vue等前端框架。只渲染变更内容，减少不必要的操作，提高渲染效率，解决渲染整个页面的问题

框架是一套完整的技术解决方案，项目选择框架后难以更改；类库提供的是单一的小功能，对项目的侵入性较小

开发环境版本<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

生产环境mini版本<script src="https://cdn.jsdelivr.net/npm/vue"></script>

三大前端框架

都适用于做单页面应用程序

Vue人气比较高，React使用的人比较多

React除了开发网站还可以开发手机APP，Vue在Weex的帮助下也可以

Vue实例

Vue实例写在js文件中

在Vue实例中有多个私有属性，分别为

el 挂载点

设置Vue实例挂载点，使实例的作用域为挂载的元素内

可以根据id、class、标签选择器进行挂载（所以可以挂在在若干个标签上），但不可以挂载到和上

data 数据对象

Vue实例的数据定义在data中

当引用这些数据时，普通数据用，数组内数据用，集合内数据用

methods 方法

有两种定义方法，类似声明式定义Func(){}，类似表达式定义Func:function(){}

方法中通过this代表Vue实例，可以获取data中的数据

directives 指令、components 组件、filters 过滤器见Vue自定义指令、组件、过滤器

生命周期函数见Vue实例生命周期

$refs

在挂载了Vue实例的元素内，在标签的属性加上ref=“xxx”，就可以在Vue实例（父标签）中用this.$.refs.xxx定位写了ref属性的标签

ref的作用是用来给元素或子组件注册引用信息，引用信息会注册到Vue实例的$refs上。

如果使用ref的是普通的DOM元素，引用指向的就是该DOM元素，Vue实例中定位这个DOM对象可以进行JS操作

如果使用ref的是子组件，引用指向的就是该子组件，Vue实例中定位这个子组件可以访问它的data和methods

Vue指令

Vue指令，作为标签的属性，属性值写在Vue实例中

v-text与v-html

v-text，设置标签的文本。属性值写在Vue实例的data中。类似直接把{ { } }写在标签的文本中，但v-text会覆盖标签文本的内容

v-html，设置标签的innerHTML，内容中有HTML结构会被解析为标签，其他与v-text一样

v-on

v-on:事件名="Func()"，v-on:可简写为@，为元素绑定事件，函数Func()写在Vue的methods里，写法Func:function(){ }

v-bind

用于绑定属性，可以是src、style、herf、class等标签属性

格式为v-bind:attr="value"，或简写为:attr="value"

如<div v-bind:style="{fontSize: size +'px'}"></div>

v-bind会把value（引号内的内容）当成JS表达式来处理。它可以是字符串（加上value外的引号一共是两层引号），可以是函数，可以是JS变量，可以是Vue实例的data中变量，可以是JS对象，可以是数组，可以是布尔值（值为false时该属性被移除）

举例v-bind:class=“”

不能用:class="c1"，Vue会去找叫做c1的方法或变量，而不是css样式

  • 字符串:class="'.c1 .c2'"，两层引号
  • 数组:class="['c1','c2']"
  • 三元表达式:class="['c0', 条件?'c1':'c2']"，或把三元表达式改造成对象:class="[{'c1':flag}, 'c2']"
  • JS对象:class="{c1:flag1, c2:flag2}"，对象的键可带引号可不带，值为布尔类型
举例v-bind:style=“”
  • 对象:style="{color:'red', 'background-color':'yellow'}"，不符合标识符规则的键需要加引号
  • 数组中包含对象:style="[{对象1},{对象2}]"

v-show

v-show="条件表达式"

满足条件时显示元素，不满足时隐藏元素

无论显示还是隐藏，元素一直存在，修改的只是css的display属性，有较高的初始性能消耗

v-if与v-else-if与v-else

用来实现条件渲染，表达式为值为真则渲染创建元素，值为假则移除这个元素，有较高的切换性能消耗

v-for

遍历数组、对象属性、基本数据范围，进行渲染

格式v-for="e in xxx"，之后可以用表示获取的元素，而xxx可以有以下几种情形

  • 数据范围v-for="i in 10"，范围是[1,10]

  • 遍历数组v-for="item in list"，list为整数数组

    可以v-for="(item,i) in list"，此时i为索引

    若数组为对象数组，item代表的是对象，可以用表示对象属性

  • 遍历对象属性v-for="(key,value) in obj"

    也可以(key,value,i)获取索引

注意，在v-for中，拥有对父作用域属性的完全访问权限。（虽然item是items中的，但是还可以使用items外的parent）

在2.2.0+版本中，使用v-for时需要指定key，使元素状态和数据关联起来，key只能是string或num

v-for="item in list" :key="item.id"，用v-bind绑定的形式使用key

v-model

只用于表单，创建双向数据绑定

<input v-model="msg">，输入文本框的值会改变Vue实例data中msg的值

v-cloak

解决插值表达式闪烁问题

设置css

并在需要的标签上加上v-cloak
