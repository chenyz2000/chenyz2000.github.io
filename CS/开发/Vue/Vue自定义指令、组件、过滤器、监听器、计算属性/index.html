Vue自定义指令、组件、过滤器、监听器、计算

Posted on 2020-08-20 | Post modified: 2020-08-27 | In Vue

自定义指令、组件、过滤器都可以进行全局注册

自定义指令、组件、过滤器、监听器、计算都是Vue实例的私有属性，只能在挂载了Vue实例的标签内部使用

directive 自定义指令

创建全局指令

指令的name定义时不需要加v-前缀，调用的时候需要加前缀

第二个参数是一个对象，对象的每个元素是钩子函数

创建Vue实例的私有指令

钩子函数

  • bind: function(){}

    执行时机为 指令绑定到元素时、此时元素还加载在内存中、未没插入HTML文件树中

    适合进行样式相关操作的初始化，只绑定一次所以只执行一次

  • inserted: function(){}

    执行时机为 元素插入到其他标签中时、即元素插入HTML文件树中时、从内存渲染到页面上时

    适合进行行为的设置，只插入一次所以只执行一次

  • update: function(){}

    所在组件的vNode更新时执行，可能会触发多次

  • componentUpdated: function(){}

    所在组件的VNode及其子VNode全部更新后调用

  • unbind: function(){}

    指令与元素解绑时调用，只调用一次

若使用的钩子函数是bind和update时，函数可以简写，等价于bind和update两个里面都写了一份代码

钩子函数参数

会自动传入四个参数，仅第一个可以修改其他都是只读的

  1. el，表示被绑定指令的元素，el是一个原生JS对象（Dom对象）

  2. binding，是一个对象，有几个属性

    • name，指令的名称，不含v-

    • expression，指令表达式的字符串。如v-func="xxx"的expression为"xxx"

    • value，指令表达式的值。如v-func="1+1"的expression为2

    • modifiers，指令使用修饰符的对象。

      oldValue，钩子函数为updated和componentUpdated时可用，是value变化前的值

      arg，传给指令的参数

  3. vnode

  4. oldVnode

component 组件

如果将所有内容都写在Vue的实例中，则它会很大

所以出现了组件，用来将Vue实例划分为不同的模块

以后需要什么功能只要调用对应的组件，提高了复用性

组件化和模块化的区别：模块化是从代码逻辑的角度划分（如Node），组件化是从UI界面的角度划分

组件有data、methods、生命周期函数，没有el属性

使用组件

使用组件时，将组件名作为HTML标签名使用（若组件名为驼峰命名则变为横线命名，非驼峰则不变）

如组件名为myComp，则在HTML中使用<my-comp></my-comp>

组件的data属性

定义方式与实例的data不同，必须是一个返回值为局部对象的方法，例如

只是data中定义方式不同，data的使用方式（插值表达式、函数里的使用）不变

data中返回的不能是全局的对象，不然组件在每一处的数据指向的都是同一个对象

组件切换

第一种方式是用v-if配合flag

第二种方式是用component标签，它会替换成JS表达式所指代的组件，JS表达式为组件名称字符串或字符串变量

父组件向子组件传值——属性绑定

默认情况下子组件不可以访问父组件的data和methods

想要访问到，就需要父组件向子组件传值

父组件在引用子组件的时候，通过v-bind的形式传递给子组件，子组件用props数组定义后可以使用

注意，props中的数据是只读的，而data中的数据是可读可写的

子组件向父组件传值——事件绑定

父组件通过事件绑定使子组件可以调用父方法，子组件使用**$emit()**调用父方法时可以向其中传递参数，因为父方法可以给父组件的数据赋值，所以相当于子组件向父组件传值

注意，@parentFunc="getSonMsg"中方法不能带括号，带括号表示把方法结果传给parentFunc，不带括号表示把方法的引用传给parentFunc

filter 过滤器

用于输出前最后的处理，可用作一些常见的文本格式化

过滤器可用在两个地方，插值表达式和v-bind表达式

使用：{ { xxx | name } }

过滤器可以串联使用，如{ { xxx | fil1 | fil2 } }

创建全局过滤器

自带一个参数，用来自动传入调用时管道符之前的值

若过滤器函数需要添加参数，定义时fuction(data, para)，调用时name(para)即可

创建Vue实例的私有过滤器

私有的优先级比全局更高

watch 监听器

可以监听指定数据的变化，然后触发watch中对应的处理函数

watch对象中元素的键就是监听的数据名，值就是对应的处理函数

处理函数可自带两个参数newVal和OldVal

监听路由地址

computed 计算属性

可以定义一些计算属性，其本质是一个方法，但在使用时是当做属性调用的

只要计算属性方法中用到的任何数据发生变化，就会立即重新计算结果值（结果值会被缓存，数据不变化时直接调用缓存值，不重新计算）

render 渲染属性

值是一个函数，有一个形参createElements，这个形参是一个方法，调用它可以把指定的组件模板对象渲染成html结构，return的结果会替换挂载了Vue实例的元素（元素整个被替换，不管内部是啥）
