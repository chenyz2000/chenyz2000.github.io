Error

  • jdbcTemplate为null。将测试程序和main所在的包保持统一，并注意写完整测试程序的注解
  • 连接mysql出现caching_sha2_password错误。默认的mysql-connector-java版本是5.7左右，连接8.0版本的mysql加密方式会出错，在pom.xml中将mysql-connector-java改为8.0以后的版本即可
  • 连接mysql出现timezone错误。在jdbc的url后加上?serverTimezone=UTC

==注解开发==

注册Mapper接口

在每个Mapper上加上@Mapper注解，或在启动类中使用@MapperScan注解扫描Mapper接口

@Mapper注解，在每一个Mapper接口上添加此注解

@MapperScan注解，指定要扫描的Mapper接口的包的路径

整合Redis

实际是在Spring Boot中用Spring Data Redis框架操作Redis

  1. 在maven工程中添加spirng-boot-starter-data-redis和commons-pools2依赖

  2. 在application.properties中配置Radis

实体类对象存入Redis

实体类需要实现序列化接口Serializable，一个对象存储起来都要实现序列化接口
