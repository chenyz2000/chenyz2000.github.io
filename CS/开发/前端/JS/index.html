JavaScript

是一门通用脚本编程语言，由浏览器编译并运行，遵循了ECMAScript标准

本身和页面动态没有关系，只有和Dom结合，才能使网页动起来

存在形式和放置位置

  • 在script标签里

  • 以文件形式存在，在script里调用

script的type属性默认值是"text/javascript"，可以省略不写

script标签可以写在head里，也可以写在body里

页面会按照从上到下的顺序运行js和显示html

所以script最好写在body里的最下面

对象

对象拥有属性和方法

JS对象是变量和方法的容器，是无序的键值对的集合

由被逗号分隔的K-V对组成，键必须为字符串。当键符合标识符定义时（只由字母数字下划线组成且以字母开头），引号可写可不写

ES6+支持对象中元素的简写，当键和值相同时，如{x:x, y:y}可简写为{x, y}

对象又像一个数组，for(key in obj)可以遍历所有键，除了obj.k1访问属性，还可以obj[k1]

正则表达式

定义

var reg = /123\d\w456/，用两个斜线括住的字符串就是正则表达式的reg

此外，/…/后面可以加任意个参数，表示不同的匹配模式：

  • g表示全局匹配

  • i表示不区分大小写

  • m表示多行匹配，只在^$时有影响

    JS正则原本就支持多行，默认模式将所有内容视为一整个字符串进行^$匹配，m模式将每一行分开进行^$匹配

匹配

  • reg.test(str)，测试str是否匹配reg，返回true或false

  • reg.exec(str)，获取str中匹配的内容，返回数组或null

    返回的数组，第一个元素为匹配到的字符串，若reg里有()进行分组，列表第二个以后的元素为分组获取到的字符串

    默认模式时，每次返回匹配的都是第一个；g全局模式时，第一次执行exec获取匹配到的第一个字符串，第二次获取第二个

  • str.match(reg)，获取匹配的字符串，返回数组，默认模式只显示第一个，g模式显示全部

  • str.replace(reg, string)，默认模式只替换第一个，g模式替换全部

    string中可以用$&表示当前匹配到的内容，$1表示匹配到所有内容中的第1个，$`表示当前匹配字符串的左侧所有文本，$'表示当前匹配字符串的右侧所有文本，$$表示$符号

  • str.search(reg)，返回匹配成功的第一个索引位置（g模式无效）

函数

函数类型

  • 普通函数

  • 匿名函数，用来作为其他函数的参数

    ES6新增了箭头函数

  • 自执行函数，声明函数并执行，可以用来做封装

    把其中的匿名函数换成箭头函数也是可以的

定义函数时，有两种办法

  • function func(){}声明式定义
  • var func = function(){}表达式定义

分别使用了以上的普通函数和匿名函数

作用域

作用域和Python类似

  • JS中无块级作用域

    花括号里的变量可以被外部访问

    注：JS6之后有新特性，let关键字可以支持块级作用域

  • JS采用函数作用域

    函数内的变量是内部形参，外部无法访问

  • JS采用作用域链

    函数嵌套时由内向外找变量，内部优先，每个函数有自己的一个作用域链

  • 作用域链在函数执行前已经创建

    浏览器解释函数时已经创建了作用域链，此时变量已经声明但未赋值

    函数写在哪里作用域就在哪里，跟返回没有关系

闭包

将变量和函数封装到函数里，外部无法获取函数内部的变量

自执行函数就是闭包
