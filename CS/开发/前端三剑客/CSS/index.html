CSS，层叠样式表

存在形式

  • 可以加在标签的style属性里，只针对一个标签

  • 也可以写在head的style标签里，针对本页面的标签

  • 更大的应用范围是将css写在专门的css文件中，只用在页面中引用该文件

优先级：宽泛的优先级低

选择器

  • 标签选择器：xxx为标签名，如div

  • id选择器：xxx为#id1，对应的标签有属性id="id1"

  • 类型选择器：xxx为.c1，对应的标签属性class="c1"

  • 层级选择：xxx为.c1 div p .c2之类的，对应的是c1类下的div中的p中的c2

  • 组合选择：xxx为.c1,.c2，对应的是c1类和c2类下的

标签选择较宽泛，id选择较狭窄，都很少使用

样式

颜色可以用white，#66ccff，rgb(0,0,0)

宽度高度：宽度有百分比和像素，高度没有百分比只有像素。百分比是相对于父标签的宽度。

盒子模型

从外到内是Margin、Border、Padding、Content

width和height设置Content的大小，background-color也是设置Content

padding:5px 10px 5px 10px设置内边距的大小，是透明的

border:2px solid red设置边框的宽度、样式、颜色

​ 边框的样式有solid、dotted、dashed、double等

​ 可以用border-width、border-style、border-color对四个边按照上右下左设置样式

margin:5px 10px 5px 10px设置外边距的大小，是透明的

最终占据的空间由四者合计

设置宽度时使用四个数字，顺序是上右下左

与其他元素默认从左上开始紧密排布

背景图片

设置背景图片需要同时设置高度和宽度，相当于给白墙上掏了一个洞，图片在洞的后面

若设置的洞过大，图片会重复出现，使用background-repeat可以使图片只出现一次

洞默认显示图片左上角，background-position是将洞不动，后面的图片进行移动，露出不同位置的图像

display

display和visibility

display:none是消失并且不保留位置

visibility:hidden是消失但保留位置

display修改行内标签和块级标签

display:inline将标签改为行内标签

display:block将标签改为块级标签

display:inline-block将标签改为可以修改宽和高的行内标签

cursor

鼠标放上去显示的样式

pointer是手，help是问号，wait是转圈，move是移动十字，crosshair是十字准星

float

使用float后标签可以无视行进行布局，但是会脱离父标签

加上<div style="clear:both"></div>可以清除float格式

position

默认值是relative

absolute，相对浏览器的窗口固定，滚动页面元素滑动；当外层为relative，则相对于有relative的外层标签固定

fixed，相对浏览器的窗口固定，如页面滑动元素不动

top

top:100px表示元素顶部与其包含块上边界的距离，同理有bottom，left，right
