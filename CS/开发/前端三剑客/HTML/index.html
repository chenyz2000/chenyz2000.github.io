超文本标记语言

超文本：使用超链接，网状结构

标记语言：标签构成的语言，如html，xml

语法

后缀名:.html或.htm，二者一样

标签种类

  • 围堵标签：包含开始标签和结束标签
  • 自闭合标签

标签属性

key:“value”

基本结构

meta

提供页面的元信息

  • 页面编码<meta charset="UTF-8">
  • 刷新和跳转<meta http-equiv="Refresh" Content="5;Url=...">Content里的数字是刷新时间，如果Content里有Url，则刷新之后跳转到该网站
  • 关键字<meta name="..." Content="...">用来给爬虫和搜索引擎使用
  • IE的兼容<meta http-equiv="X-UA-Compatible" Content>="IE=edge"
link
  • 引入css<link rel="stylesheet" type="text/css" href="...">
  • 使用icon作为标题的图片<link rel="shortcut icon" href="...">

body内标签

特殊字符

https://blog.csdn.net/u013778905/article/details/53177042

分类

  • 块级标签：自己占一行，有高度和宽度。如div，h1，p
  • 行内标签：有多少内容占多少，无高度和宽度。如a，span，select

p和br标签

p是一个段落，自动换两行

<br>是换行符

a标签

<a href="...">显示的内容</a>a标签href属性的跳转功能是特有的

  • 跳转：href里可以是url，跳转到网页

  • 锚：也可以是#id，跳转到当前页面对应id的标签处

    html中id值唯一

a标签有特有属性target，target="_blank"表示跳转时打开新的标签页

h标签

从h1到h6

html的样式默认情况下如此，不过它们都可以被css自定义

div和span

本身没有特殊效果，但是可以修改其中内容的样式

input

单行文本框<input type="text"/>

密文<input type="password"/>

多行文本框<textarea>...</textarea>

单选圆钮<input type="radio"/>，想要多个互斥可以在每个标签里加name="..."属性且值相同

复选框<input type="checkbox"/>，可以通过cb.checked的true/false（是obj.property不是attribute）操作选中与不选

选择文件<input type="file" />

按钮<input type="button" value="" />，空按钮，默认无功能

select

选择下拉框

option中可以加属性size="5"表示下拉框打开后显示几个选项，属性multiple表示可以多选

form标签、嵌套按钮和表单提交

submit和reset可以对当前form表单里填写的内容进行控制，对input、select、textarea都有效

  • <form action="url" method="get">

    action中填写提交到的url，不能带参数，默认为当前页面

    method是提交方式，默认为get，还有post方式

  • 想要服务器获取数据，需要加name属性，文本框提交的数据格式是name:text的字典

  • 表单中的互斥单选圆钮name属性相同，只要有其中之一勾选，提交的数据就为name:on；若想区分，则给每个圆钮加value="..."的属性，提交的数据为name:value

  • 进行文件提交，服务器获得的是name:文件名，而不是文件，需要在form中加入属性**enctype="multipart/form-data"**才会分块上传

  • select中，name是select标签的属性，value是option标签的属性

提交表单的顺序：

先根据form表单中enctype属性对数据进行编码，再根据form表单中的action和method属性向指定的地址发送数据

其他通过form表单将数据传递到后端的笔记，见《前后端分离》

其他

image

<img src="...">插入图片

Lable

点击lable时会触发关联id的元素

ul，ol，dl

序号，ul是圆点，ol是数字，dl中dt是顶格dd是缩进

table

thead和tbody可以省略，只要tr套th或td一行一行缕下来就行

table属性border="1"显示表格线

合并单元格：横向合并td属性clospan="2"，纵向合并td属性rowspan="2"，还要把多余的格删除

fieldset

<fieldset></fieldset>扩住的部分外围会有一个边框

iframe

<iframe src="url"></iframe>将其他网页展示在本网页
