JavaIO

Posted on 2020-05-28 | Post modified: 2020-05-28 | In Java基础

文件系统由OS管理，与Java进程是两套系统

Linux文件目录分隔符是/

Windows是\，在程序中要用/或\\代替

文件和目录类File

File类不涉及文件内容，只涉及文件属性

文件包括文件内容和文件基本属性，基本属性包括名称、大小、扩展名、修改时间等

可调用的方法有：创建删除文件、获取路径、判断是目录、文件大小、获取子文件列表等

File类和OS无关，但会受到OS的权限限制

JavaIO

Java读写文件，只能以数据流的形式

因为文件可能很大，不能全部放入内存，而是要使用流分批读写

JavaIO包含有节点类和包装类，包装类含有转化类和装饰类

节点类对文件进行读写，转化类对字节、字符、数据类型之间进行转化，装饰类对节点进行装饰

字节是最基础的存储单位，文件都是以字节进行保存

节点类

对文件进行读写

字节流：InputStream, OutputStream => FileInputStream, FileOutputStream

字符流：Reader, Writer => FileReader, FileWriter

转化类

读取时字节转字符：InputStreamReader

写入时字符转字节：OutputStreamWriter

装饰类

封装数据流，可以直接以int,double输入输出： Data+字节两个

在内存开缓存区，提升读写速度：Buffered+字节字符四个

文件

文件类型

  • 文本文件，如txt
  • 二进制文件，如数据文件dat、图片文件
  • 特殊格式的文本文件，如xml
  • 特殊格式的二进制文件，如doc

写文件

BufferedWriter(OutputStreamWriter(FileOutputStream))

写字符到缓存区（字符到字节转化（写字节））

操作流程：打开文件，逐行写入，关闭文件

方法有write()和newLine()

关闭只需要关最外层的流

读文件

操作流程：打开文件，逐行读取，关闭文件

方法有readLine()

try-source语法糖可以自动关闭流
