Linux基本命令

Posted on 2020-02-25 | Post modified: 2020-07-30 | In Linux

nohup

nohup是非挂断运行（不加&还是前台）

之后通过exit退出ssh连接即可非挂断运行

如果需要查找命令以及停止

screen

部分情况下nohup不起效，可以采用更高级的screen命令

screen -S命令是启动一个新窗口

在窗口中按ctrl+a，之后按d即可暂离窗口
