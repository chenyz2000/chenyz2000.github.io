Git分支命令

Posted on 2020-07-13 | Post modified: 2021-11-24 | In 工具

远程操作命令包括clone、remote、fetch、pull、push

remote

git remote -v列出所有远程仓库和url，其中-v是–verbose的意思，用-vv结果一样

git remote add <name> <url>

git remote rm <name>

git remote prune origin删除本地已拉取的 但远程已经删除的分支

git prune是删除本地悬空的提交（如因为reset造成的）

fetch和pull

git fetch获取远程分支的更新，只更新origin远程分支，不改变本地分支

只是获知更新，之后需要手动merge远程分支到当前分支

git pull相当于git fetch; git merge origin/main

git pull --rebase等于git fetch; git rebase origin/main

push

将本地分支推送到远程仓库，并更新origin分支

git push <name> <branch>

追踪关系

在本地分支与远程分支之间，有一种追踪关系（tracking）

追踪关系用来指明push、pull的对象

clone时、根据远程分支创建本地分支时会自动进行追踪
