IDEA

Posted on 2020-08-09 | Post modified: 2021-08-12 | In Tools

快捷键

ctrl+

w 选中代码块（逐渐扩大） <==> ctrl+shift+w 逐渐减小

e 最近文件

j 自动代码片

y 删除一行

o 重写接口或父类方法

F12 显示方法和成员变量

~ 调整View

b 查看被调用的地方

d 向下复制一份

alt+

~ 版本控制

j 选中同名变量

insert 再送生成getter、setter、toString等

F1 多个功能的select in面板

ctrl+shift+

j 与下一行合并

i （缩略）查看定义

u 全变成大写或小写

- 收缩所有代码段

ctrl+alt+

u 类图简略

o 调整import

v 提取语句中的变量出来

l 格式化

其他

按下滚轮动鼠标 多光标

插件

Restful Toolkit：用ctrl+\搜索url对应的类

Codeglance：侧边代码

插件中心网络连接不上：原本的代理方式是手动，无法连接；之后改为自动，也无法连接；最后改为不代理，就可以连接了

设置

show memory inidcator：显示占用内存

show quick documentation on mouse move：开启悬浮提示（相当于是自动的ctrl+q）

File Encoding：一共修改三处编码为UTF-8

show tabs in one row：取消可以使标签页tab多行显示
