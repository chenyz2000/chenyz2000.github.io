Java反射

Posted on 2021-10-06 | Post modified: 2021-10-07 | In Java SE

注解、反射是Spring系列、MyBatis等框架的底层机制

反射Reflection

java.lang.reflect包

Java是静态语言，但反射让Java有了一定的动态性，可以称为准动态语言

不过反射的速度会慢，例如用反射创建对象的速度比new慢几十上百倍

Class对象

Class类由系统掌控，Class对象由系统创建

每个类加载完后，JVM的堆中都会对应地产生一个Class对象

每个类对应唯一的Class对象，一个Class对象对应加载到JVM中的一个.class文件

除了类以外，内部类、接口、数组、定义的注解、enum、基本数据类型、void都可以有Class对象

元素类型相同、维度相同的数组，无论长度多少，对应的都是同一个Class对象

Class对象包含了类的完整结构信息（字段、方法、构造器、父类、接口、注解等），可以在程序执行期间获取，并操作对象的属性、方法和构造器

setAccessible()设为true可以避免安全检查，可以提高反射的效率，大约三倍

通过反射动态创建对象

  • c.newInstance()：调用无参构造函数，返回创建的对象，Object类型
  • c.getSuperClass()：获得当前类的父类的Class对象
  • c.getInterfaces()：获得当前类实现的接口的Class对象列表
  • c.getClassLoader()：获得当前类的类加载器

Method、Field、Constructor都是java.lang.reflect包中的

反射操作泛型

Java中的泛型仅是给编译器javac使用的，用以确保数据的安全性和免去强制类型转换

Java采用了泛型擦除的机制，一旦编译完成，所有和泛型有关的类型全部擦除

为了使用反射操作泛型，Java新增了ParameterizedType、GenericArrayType、TypeVariable、WildcardType几种类型来代表不能被归一到Class类中但又和原始类型齐名的类型

  • ParameterizedType：表示参数化类型，如Collection< String >
  • GenericArrayType：表示一种元素类型是参数化类型或者类型变量的数组类型
  • TypeVariable：各种类型变量的公共父接口
  • WildcardType：代表一种通配类型表达式

这四个类型是Type类的子类

Method对象有getGenericParameterTypes()方法，返回Type数组，获取参数

和getGenericReturnType()方法，返回Type对象，获取返回值
