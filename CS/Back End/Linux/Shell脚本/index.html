Shell脚本

Posted on 2021-06-27 | Post modified: 2021-06-27 | In Linux

shell介绍

种类

  • Bourne shell（sh）

  • C shell（csh）

  • Korn shell（ksh）

  • Bourne Again shell（bash）

脚本执行方式

假设脚本名为script.sh

  • bash < script.sh，输入重定向，会产生新进程
  • bash script.sh，会产生新进程
  • · script.sh，在当前shell中执行（·表示当前shell），不会产生新进程
  • 将脚本设置为可执行，之后./script.sh直接执行

前导知识

通配符

  • *：任意个字符

  • ?：单个字符

  • [abc]：括号内的任一字符

  • [!abc]或[^abc]：不在括号内的字符

  • {str1,str2}：匹配括号内的任意字符串，可以用..表示范围

引号

  • 单引号：硬转义，所有字符作为普通字符

  • 双引号：软转义，除特殊字符（$`\）外都作为普通字符

  • 反引号：命令替换，等价于$(命令串)，将命令运行后的结果填在当前位置，不视为字符串

    反引号可以嵌套使用，不过内层的需要加\转义

重定向

将输入输出文件的方向从标准IO（键盘屏幕）重定向到命令、文件

  • <文件输入到命令

  • >命令输出到文件

  • >>追加输出到文件

  • <<即时文件输入，表示从屏幕上即时输入到参数，符号右侧格式为!!xxx!!

文件描述字重定向：

标准输入文件stdin，标准输出文件stdout和标准出错输出文件stderr的文件描述字分别为0，1和2

使用格式为命令 描述字> 文件 或命令 描述字>> 文件

命令执行操作符

  • 命令写在同一行，用;分隔
  • 管道符命令1|命令2，将命令1的结果作为命令2的输入

逻辑与、逻辑或

  • 逻辑与命令1 && 命令2，命令1执行成功才执行命令2

  • 逻辑或命令1||命令2，命令1执行不成功才执行命令2

成组命令（视为一个命令）

  • (xxx)，产生新进程
  • { xxx;}，注意需要空格和分号，不产生新进程

位置参数

$0：命令名

$1, $2, …, ${10}：后面的参数

位置参数只能引用，不能用一般的方式赋值，可以用set命令赋值

shift命令移动位置参数，$1之后的参数每次左移一位，$0不变

预先定义的特殊变量

  • $#命令参数的个数，不包含命令名本身

  • $?上一条命令执行后的返回值，是一个十进制数

  • $$当前进程的进程号

  • $!上一个后台命令对应的进程号，这是一个由1~5位数字构成的数字串

  • $-由当前shell设置的执行标志名组成的字符串

  • $*命令行中实际给出的所有实参字符串，它并不仅限于9个实参

  • $@与$基本功能相同，即表示在命令行中给出的所有实参。但“$@”与“$”不同

环境变量，用$xxx直接引用

  • HOME：用户主目录的全路径名
  • LOGNAME：注册名，由Linux自动设置
  • MAIL：系统信箱的路径
  • PATH：shell从中查找命令的目录列表。可以这样设置PATH=$PATH:$HOME/bin
  • PWD：当前工作目录
  • SHELL：你当前使用的shell
  • TERM： 终端类型

shell编程

变量

都是字符串型，无其他类型，大小写敏感

引用变量时用$xxx，或${xxx}，若后面跟着字符只能用带大括号的否则会有歧义

数组

支持一维数组，编号从0开始，数组不限大小

定义：arr=(val1, val2, ..., valn)

赋值：arr[0]=val

引用：${arr[1]}，不写下标${arr}时默认下标0

arr[*]或arr[@]表示全部元素

输入输出命令

输入：read 变量，将之后读取的值赋给指定的变量

输出：echo 字符串或变量，选项-e表示之后可以有转义字符

算术运算

((算术表达式))或let "算术表达式"，但是只有$((算术表达式))才能把值赋出来

算术表达式中直接利用名称访问命名的参数，不需要带有$符号

如((j=i*6+2))，j=$((i*6+2))

条件表达式

有以下三种写法

  • test 表达式
  • [ 表达式 ]
  • [[ 表达式 ]]，比单中括号多了支持正则表达式

注意：中括号内侧一定要有空格！

test可以对文件、字符串、数值进行测试

test数值测试

参数 说明
-eq 等于
-ne 不等于
-gt 大于
-ge 大于等于
-lt 小于
-le 小于等于
