Python os库

Posted on 2021-06-22 | Post modified: 2021-06-22 | In Python

os.path子模块

即如os.path.exists()，可以通过from os import path导入此模块

  • exists(path)，其中path为文件或目录路径字符串

  • abspath(path)，获得文件绝对路径

  • isfile(path)，isdir(path)

  • getsize(path)，返回文件大小，以字节为单位

  • basename(path)返回不含目录的文件名，dirname(path)返回不含文件名的目录名，split(path)返回目录名和文件名的二元组，splitext(path)返回路径加文件名和扩展名的二元组

  • join(path, [path1, …])合并路径

os目录操作

  • getcwd()，返回当前工作目录

  • chdir(path)，切换工作目录

  • listdir(path=“.”)，默认值为当前目录，返回包含所有文件和子目录名称的列表

  • walk(path)，递归遍历目录，返回一个迭代器，其中每个元素是针对一个目录的(dirpath, dirnames, filenames)三元组

  • mkdir(path, mode=0o777)，makedirs(path, mode=0o777)，创建目录和递归创建目录，默认权限为777

  • rmdir(path)，removedirs(path)，删除空目录和递归删除空目录，不存在或不为空都会异常

  • remove(path)，删除文件，path必须为文件否则异常

os操作环境变量

  • os.environ[“Path”]，从一个字典里得到值，不仅可以获得还可以修改环境变量
  • os.getenv(“Path”)，获得PATH的环境变量

os.system(“cmd”)，执行命令，阻塞当前进程并启动子进程，直到执行完成。其返回值为int，该值右移8位即等于在linux中运行的返回码。容易出错，复杂的命令尽量不要使用
