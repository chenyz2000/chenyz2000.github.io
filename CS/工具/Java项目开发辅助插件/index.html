注解

==@Data==：最全面，注解在类上，提供所有属性的get和set方法，以及equals、canEqual、hashCode、toString方法

@Builder：注解在类上，提供builder方法进行对象创建，调用方法例如

其中，第一行和第三行是固定的，第二行是类的属性

@Getter，@Setter：注解在类或者属性上

@ToString，@EqualsAndHashCode：注解在类上，提供toString方法或equals、canEqual、hashCode 方法

@NoArgsConstructor，@AllArgsConstructor：注解在类上，提供一个无参/全参的构造方法

@Slf4j：日志注解，注解在类上，省略了Logger对象的创建方法（就那一行），在类中直接用log调用debug()等方法即可
