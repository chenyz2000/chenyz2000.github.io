快捷键

重要快捷键

ctrl + alt + L：格式化代码

ctrl + W：扩大选中的代码块，再加shift是缩小

ctrl + D：向下复制当前行

ctrl + Y：删除当前行

ctrl + O：重写、实现父类或接口的方法

alt + insert：生成getter、setter、toString等

ctrl + alt + O：调整导入的包

ctrl + alt + T：为当前代码选定环绕代码(if、for、try等)

ctrl+shift + del：删除环绕代码

ctrl+

==w <-> ctrl+shift+w 逐渐减小==

e 最近文件

j 显示所有代码片的简写

y ==删除一行==

o ==重写接口或父类方法==

F12 显示方法和成员变量

~ 调整View

b 查看被调用的地方

==d 向下复制一份==

alt+

~ 版本控制

j 选中当前变量下次出现处 alt+shift+j选中前一个 ctrl+shift+alt+j选中文件中所有出现

insert 生成getter、setter、toString等

F1 多个功能的select in面板

ctrl+shift+

j 与下一行合并

i 查看简略的定义

u 全变成大写或小写

- 收缩所有代码段

ctrl+alt+

u 类图简略

o 调整import

m 提取语句中的方法出来

l 格式化

t为选定的代码加一段代码进行环绕(if、for、try等)，ctrl+shift+del可以删除环绕代码

其他

按下滚轮动鼠标 多光标

插件

Restful Toolkit：用ctrl+\搜索url对应的类

Codeglance：侧边代码

插件中心网络连接不上：原本的代理方式是手动，无法连接；之后改为自动，也无法连接；最后改为不代理，就可以连接了

设置

show memory inidcator：显示占用内存

show quick documentation on mouse move：开启悬浮提示（相当于是自动的ctrl+q）

File Encoding：一共修改三处编码为UTF-8

show tabs in one row：取消可以使标签页tab多行显示
