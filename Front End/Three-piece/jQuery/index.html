jQuery

Posted on 2020-07-23 | Post modified: 2020-07-25 | In Three-piece

jQuery

类库，封装了Dom和JS，不用担心兼容性的问题

简化原生JS的写法，对Dom进行操控

2.x版本不兼容低版本IE了，使用1.12.x等版本

通过CDN导入jQuery

jQuery.xxx简写为$.xxx

Dom对象转为jQuery对象：$(obj)

jQuery对象转为Dom对象：$(…)[0]

jQuery对象可以赋给var定义的变量，建议变量名以$开头以示区分

使用时查看文档

选择

选择器

$(“#id”)，ID选择器

$(“tag”)，标签选择器

$(“.c1”)，类选择器

$(“.c1 > .c2”)，层级选择器，>为子元素，空为后代，~为兄弟

$(“#id1, .c1”)，组合选择器，A或B成立都可以

选择器内筛选

写在$(“xxx”)的引号中，xxx的后面，xxx可以为空

  • :not(:selector)

  • :first第一个，:last最后一个，，:even和:odd

  • :eq(n)第n个标签，:gt(n)序号大于n，:lt(n)序号小于n

  • 属性筛选——[attribute=“…”]，可以写多组，以及有不等于、包含字符等选项

  • 子元素——:first-child，:nth-child等，是其父标签的第n个孩子

  • 针对表单使用——表单类型，:input，:text，:submit，:button等

    ​ 表单属性，:checked，:selected，:disabled

    $(“:text”)和$(“input[type=“text”]”)等价

筛选器

对选择器结果进行的另一层筛选

$(“…”).xxx其中xxx为筛选器

  • 过滤：eq()索引，first()，last()，hasClass()，is(标签类型)

  • 查找：children()子标签，find()后代

    ​ next()下一个兄弟，prev()，siblings()

    ​ parent()父标签，parents()祖先

操作

jQuery对选择到的所有标签可以统一进行处理，不需要像在Dom中循环

根据参数不同，各个函数可以获得或设置属性的值

一般有两种，无参获取有参设置，或一个key参数获取两个key-value参数设置

HTML/文本

  • 获取或设置对象内容：html()，text()
  • 删除：remove()同时删除绑定的事件，detach()不影响绑定的事件
  • 内部删除：empty()
  • 内部插入：append()内部最后，prepend()内部最前
  • 外部插入：after()，before()

clone()返回和原有对象一样的jQuery对象，可以和其他对象一样调用方法，适用于复制一部分标签插入到html中

Attribute属性

  • 属性：attr(“属性名”)获取值，attr(“属性名”, 值)赋值，removeAttr(“属性名”)删除属性；

prop同理，prop可用在如checkbox的checked属性

  • class属性：addClass()，removeClass()，toggleClass()有就移除没有就添加
  • value属性：val()获取或设置

CSS样式

  • css样式/style属性：css(“样式名”, “值”)，如DOM中tag.style.fontSize(“18px”)改为$(“…”).css(“fontSize”, “18px”)

  • 位置：

    • offset()距离窗口（不是可视区域）顶部和左边的距离；position()距离父标签（不是position:relative祖先标签）顶部和左边的距离
    • scrollTop()上下滚动条位置；scrollLeft()左右滚动条位置

  • 尺寸：

    • height()，width()，包括边框

  • innerHeight()，innerWidth()，不包括边框

    • outerHeight()，outerWidth()，默认包括边框，参数为false则不包括

each

若操作为自定函数，$(“…”).each(function(){})，函数可用实参i表示索引，函数体中可用$(this)表示当前对象

若在函数体内写return，只会return循环的当前次，后面的循环还继续，函数体中的return相当于each循环中的continue

若在函数体内写return false，则该次循环之后的循环都会被终止，相当于each循环中的break

事件

建议使用jQuery对象绑定事件，而不是在Dom对象属性中写onclick=“”

格式：$(“…”).click(function(){})，等价于$(“…”).bind(“click”,function(){})

在函数体中可以使用$(this)指代当前jQuery对象

方法有click()，

文档加载完毕，自动执行

HTML文档树加载完毕（资源可能未加载完）时自动调用函数

$(document).ready事件可以定义多个方法，每次定影都是向其中追加，而不是覆盖

适用于为html添加样式、绑定事件等，要注意onclick等DOM事件的函数不能写在它内部，不然作用域不同会找不到函数。

它的内部绑定的事件只会在文档加载完毕后就加载完毕，不会在之后的html修改后还能绑定，若想要延迟这个加载，可以利用委托进行延迟加载，当使用时才会进行绑定而不是一开始绑定，$(“”).delegate(“选择器”, “click”, function(){})

在HTML元素和资源都加载完毕时调用函数

window.onload事件每次只能保存对一个函数的引用，它会覆盖掉之前的函数，所以不能再现有行为上添加新的行为。

  • [ ] 在script标签顶部写？

事件链

jQuery只要在函数体中加return false，则可以阻止后面的事件执行

扩展方法

为了方法不冲突，很多包都用(function(jq){ })($);自执行函数将方法包裹起来，并传入jQuery作为参数

其中，$.extend()方法的作用是：

可以接收一个或多个参数，若接收多个时，是将后面所有对象合并到第一个对象（修改第一个对象并返回）；若接收一个时，则将该对象合并到jQuery对象中

所以可以用它将方法合并到jQuery中

所以不使用extend也可以扩展方法
