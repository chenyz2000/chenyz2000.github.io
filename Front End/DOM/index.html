DOM

Posted on 2020-07-21 | Post modified: 2020-07-22 | In Front End

DOM

文档对象模型，Document Object Model，一种用于HTML和XML文档的编程接口，将文档作为文档树

查找

直接查找

var t = document.getElementById()，根据ID获取一个标签

var t = document.getElementsByName()，根据name属性获取标签集合

var t = document.getElementsByClassName()，根据class属性获取标签集合

var t = document.getElementsByTagName()，根据标签名获取标签集合

间接查找

节点查找

t.parentNode，父节点

t.childNodes，所有子节点

t.firstChild，第一个子节点

t.lastChild，最后一个子节点

t.nextSibling，下一个兄弟节点

t.previousSibling，上一个兄弟节点

标签元素查找

t.parentElement，父节点标签元素

t.children，所有子标签元素

t.firstElementChild，第一个子标签元素

t.lastElementChild，最后一个子标签元素

t.nextElementtSibling，下一个兄弟标签元素

t.previousElementSibling，上一个兄弟标签元素

区别

节点比标签元素多了文本内容

操作

对内容操作

t.innerText内部仅文本内容

t.innerHTML内部HTML内容（子标签+文本）

t.value文本框的值

对属性操作

t.getAttribute(“属性名”)获得属性值

t.setAttribute(“属性名”, “属性值”)进行修改

t.removeAttribute(“属性名”)删除

对class操作

t.className获取class属性的值，返回字符串

t.classList获取class属性的值，返回列表

t.classList.add()

t.classList.remove()

对样式操作

t.style.color=”red”

标签的创建与添加

优先使用对象形式

对象形式

var tag = document.createElemnt(“a”)创建

再对该对象的属性进行修改，注意一些含连接符的attribute变成了大写的property，如font-size变成fontSize

获取添加tag进入的对象obj

obj.appendChild(tag)，默认是追加在最后

obj.insertBefore(tag, obj.children[0])

obj.insertAdjacentElement(“beforeEnd”, tag)

字符串形式

var tag=”<a>abc</a>”

获取添加tag进入的对象obj

obj.insertAdjacentHTML(“…”, tag)保留标签

obj.insertAdjacentText(“…”, tag)添加纯文本

第一个参数只有四个值”beforeBegin””afterBegin””beforeEnd””afterEnd”

Begin和End指的分别是obj的开始标签和结束标签

位置操作

clientHeight，可见区域height + padding高度

clientTop，可见区域border高度

offsetHeight，可见区域height + padding + border高度

offsetTop，距离祖先标签中最近的定位标签的高度

scrollHeight，全文height + padding高度

scrollTop，滚动条滚动了多少

body和文本框都可以有滚动条，document.body.scrollTop=0可以让页面返回顶部

在文本框中，若文本超过文本框大小，部分可见，可见的部分是可见高度，全文是全文高度

提交表单

不用submit按钮，用对象form对象的submit()

当前页面

location.href()获取或修改当前网址

location.reload()重新加载

其他

console.log()

alert()

confirm()，给出确定和取消两个按钮，这个函数有返回值true/false用于分辨

定时器

setInterval(“代码语句”或匿名函数,1000)，设置多次定时器

clearInterval(setInterval返回的特征值)，删除多次定时器

setTimeout()，设置单次定时器

clearTimeout()，删除单次定时器

绑定事件

事件

onfocus，获取焦点

onblur，丢失焦点

onclick，单击

ondblclick，双击

onscroll，滚动

onchange，内容改变

onkeypress，按下并松开键盘

格式

在标签的属性里写onfocus=”Focus()”并在script中写Focus()函数的代码，或在JS脚本里写window.onkeypress=function(){}

特殊参数

实参this表示当前标签

实参event表示事件对象

默认事件

先执行自定义，再执行默认事件：a、form的submit

先执行默认事件，再执行自定义：checkbox

事件链

若有多个事件，改写onclick=”return Func()”，当onclick事件的返回值为false，则阻止后面的事件发生

此功能适用于表单信息的验证
